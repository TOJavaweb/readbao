“一元惊喜”指用户花费1元购买“闪电网盘”1M空间后即可获赠1个闪币，用户可使用闪币参与奖品抽奖的服务。奖品分配对应数量的号码，每个号码价值1个抢币，当一件奖品所有号码售出后，根据预先制定的规则计算出一个幸运号码，持有该号码的用户，直接获得该奖品。

声明：所有奖品抽奖活动与本公司无关。
 夺宝规则

1、获得闪币：用户花费1元购买闪电夺宝网盘空间，即可获赠1个闪币

2、挑选喜欢的商品：奖品分配对应数量的号码，1个闪币可以获得其中1个号码（系统随机分配）

3、获得奖品：当所有号码都被分配完毕后，系统根据规则计算出1个幸运号码，持有该号码的用户，直接获得该奖品。

 幸运号计算规则

1、截止该奖品开奖时间点前本站全部奖品的最后50参与时间所代表数值之和；

2、将这50个时间的数值进行求和（得出数值A）（每个时间按时、分、秒、毫秒的顺序组合，如20:15:25.123则为201525123）；

3、数值B为奖品所需人次；

4、数值A除以数值B得到的余数+原始数10000001，得到最终幸运号码，拥有该幸运号码者，直接获得该奖品。

5、余数是指整数除法中被除数未被除尽部分， 如7÷3 = 2 ......1，1就是余数 。
